# Ezan Vakti

Ezan Vakti mobil uygulaması için hazırlanmış gösterim / vitrin reposu.

## 📌 Hakkında

Bu repo, Türkiye'deki namaz vakitleri ile ilgili modern bir kullanıcı arayüzü ve ayarlanabilir alarm seçenekleri sunan bir mobil uygulama örneğidir.

## 🚀 İçindekiler

- `App.tsx` - Uygulamanın temel bileşeni
- `components/` - UI bileşenleri
- `services/` - Vakit hesaplama ve alarm servisleri
- `public/` - Statik varlıklar
- `screenshots/` - Görsel tanıtım görüntüleri

## 🧩 Teknolojiler

- React + TypeScript + Vite
- Capacitor desteği ile mobil platforma hazır yapı
- Modern CSS ile kullanıcı arayüzü

## 📦 Kurulum

1. Repo klasörüne gidin:
```bash
git clone <GITHUB_REPO_URL>
cd ezan-vakti
```

2. Bağımlılıkları yükleyin:
```bash
npm install
```

3. Geliştirme modunda çalıştırın:
```bash
npm run dev
```

## ℹ️ Notlar

- Bu repo, sadece sunum amaçlı hazırlanmıştır.
- Mevcut kodlar web ve mobil odaklıdır; Android native yapılandırma için ekstra adımlar gerekebilir.
